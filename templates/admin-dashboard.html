About This System

This online proctoring system is designed to maintain academic integrity in online examinations. It allows administrators to create and assign tests to students, and monitor them in real-time using advanced AI models such as YOLOv8. The system flags suspicious behaviors, provides live video feeds, and stores activity logs. With email notifications and performance analytics, it ensures a secure and transparent exam environment.
